## 🌟 Architecture Overview

The system is compartmentalized into discrete intelligence engines:

### 1. Sovereign Fund Dashboard (`/sovereign-fund-dashboard`)
The **Ultimate Command Center** serving as the visual front-end for the entire network. Built with React (Vite) and Tailwind CSS using a specialized "Living Ledger" dark/gold theme.
* **Alpha Heatmap**: Tracks high-beta assets, aggregating Short Interest %, Sentimental News, Retail behavior, and mathematical "Crowdedness."
* **Agentic 13F Ledger**: Projects "Copy-Cat Alpha" by predicting forward equity positions of major institutional hedge funds.
* **Governor Console**: A real-time, interactive stream where AI pods debate market conditions. The Governor orchestrates the Sentiment Pod (Newsflow), Short-Interest Oracle (Quantitative), and Mirror Pod (13F logic) before authorizing execution.

### 2. Autonomous War Room (`/war_room/agentic_terminal.html`)
An interactive terminal interface designed specifically to handle system safety interlocks and display synthetic confidence matrices. Tests human-in-the-loop oversight workflows against conflicting multi-agent LLM consensus.

### 3. Quantitative Trading Engine (`/trading_engine`)
The backend laboratory containing raw algorithmic DNA:
* Bollinger Band Breakout models.
* RSI Momentum configurations.
* Mean-reversion statistical arbitration pairs.

## 🚀 Quickstart

To run the interactive Command Center locally on your machine:

1. **Start the Oracle Backend** (Serves the heatmap, news API, and debate logics):
   ```bash
   cd sovereign-fund-dashboard
   node server.js
   ```

2. **Boot the React Frontend Application**:
   Open a separate terminal window:
   ```bash
   cd sovereign-fund-dashboard
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173/` to oversee the simulation.
